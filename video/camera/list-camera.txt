camera_control


camera_control command line options:
    -a <trace options>  : Ada_Lib library trace options (Ada_Lib)
    -c <directory>      : current directory             (Camera Lib)
    -E                  : simulate camera               (Camera Lib)
    -G <trace options>  : GNOGA traces                  (GNOGA library)
    -h                  : this message                  (Ada_Lib)
    -l                  : local camera                  (Camera Lib)
    -P                  : pause                         (Ada_Lib)
    -r                  : remote camera                 (Camera Lib)
    -t <trace options>  : trace options                 (Camera Lib)
    -T <trace options>  : Camera.Lib.Options
    -v                  : verbose                       (Ada_Lib)
    -V <trace options>  : trace options                 (Video Lib)
    -w                  : Web server port               (GNOGA library)
    -@?                 : trace test condition          (Ada_Lib)
    -@i                 : indent trace                  (Ada_Lib)
    -@p                 : include program in trace      (Ada_Lib)
    -@t                 : include task in trace         (Ada_Lib)
    -@x                 : exclude time in trace         (Ada_Lib)
